# 📁 项目目录结构说明

## 组织结构

```
📁 项目根目录/
├── 📜 docker-compose.yml          # Docker Compose 配置文件
├── 📜 .env                        # 环境变量配置文件
├── 📜 .env.example               # 环境变量模板文件
├── 📜 .gitignore                 # Git 忽略文件配置
├── 📁 backend/                   # 后端代码
│   └── 📁 MyCampus/              # Spring Boot 应用
├── 📁 frontend/                  # 前端代码
├── 📁 scripts/                   # 🆕 脚本目录
│   ├── 📜 backend-deploy.sh              # 部署脚本
│   └── 📜 backend-check-env.sh           # 环境检查脚本
├── 📁 docs/                      # 🆕 文档目录
│   ├── 📜 DOCKER-DEPLOYMENT.md   # Docker 部署指南
│   ├── 📜 ENV-CONFIG.md          # 环境配置说明
│   ├── 📜 开发规范.md             # 开发规范（原有）
│   └── 📜 NABCD.md               # NABCD 分析（原有）
└── 📁 mongo-init/                # MongoDB 初始化脚本
    └── 📜 init-mongo.js
```

## 🚀 快速使用

```bash
# 环境检查
./scripts/check-env.sh

# 完整部署
./scripts/backend-deploy.sh
```
