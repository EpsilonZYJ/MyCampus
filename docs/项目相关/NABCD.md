# NABCD

## Need

在华科大，许多同学既想吃食堂实惠可口的饭菜，又不想忍受高峰期排队和往返的奔波。同时，也有一部分同学希望利用空余时间通过简单的劳动赚取零花钱。我们的APP旨在搭建一个连接这两类同学的校园内部平台，一方面通过智能菜品搜索(输入想吃的食材然后进行推荐？)帮助同学快速找到心仪美食，另一方面通过校内众包跑腿模式，让同学能“懒在寝室吃食堂”，让跑腿同学能“顺路捎带赚点钱”，最终打造一个高效、经济、可信的校园餐饮服务新体验。

## Approach

平台前端采用react.js框架，后端采用java springboot框架。对于用户上传的菜品以及跑腿信息发布上传到后端进行数据处理后存入数据库，数据库我们打算使用MongoDB。

## Benefit

对于想要用餐的同学，可以高效挖掘食堂菜品、确定今日菜单。通过智能菜品推荐功能，用户只需输入食材、口味或菜名关键词，系统便能从各食堂中精准匹配并推荐心仪美食，有效解决“选择困难症”。解放了用户的时间与精力，让大家足不出户就能享用食堂可口且实惠的饭菜，省去了高峰期排队和往返奔波的辛苦，真正实现了“懒在寝室吃食堂”的便捷体验。对于希望利用空余时间的同学，平台则提供了一个安全、可靠的灵活增收渠道。跑腿同学可以像“顺路捎带”一样接取订单，将零碎时间转化为实际收益，整个过程简单、自由且极具效率。平台通过连接这两类需求，不仅优化了校园内的资源分配，减少了食堂拥堵，更营造了一种互助、便捷的校园生活新氛围，为所有用户创造了一个高效、经济且值得信赖的新体验。

## Competitors

目前在校园内，存在以部分互联网技术社团的产品已经解决了食堂美食推荐。而对于跑腿，一方面与美团等商业化产品有功能重叠，另一方面部分QQ群等充当了这一功能。但本平台优势在于前者并没有一个集成化的校园生活产品；而后者要么商业化比较严重，要么没有统一的平台，对于美团等外卖平台仅局限于商家以及食品，并不能配送食堂餐饮，也不能代取快递或是其它跑腿内容，对于QQ群可以帮忙跑腿但是并没有统一的平台并且缺少平台监管而带来一些不便。而该平台也可以从单个学校开始做起，在推广后可以覆盖更多高校。

## Delivery

通过和校内影响力较大的一些学生组织合作进行推广，从而可以让更多的同学参与到其中来。
