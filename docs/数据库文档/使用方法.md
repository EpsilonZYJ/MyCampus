# MyCampus MongoDB 初始化工具 - 使用指南

## 快速开始

### 方式 1: 使用快速启动脚本（推荐）

```bash
cd mongo-init/local-deploy
./quick-start.sh
```

脚本会自动：
- ✅ 检查 Node.js 和 MongoDB 环境
- ✅ 安装必要的依赖包
- ✅ 提供交互式菜单选择初始化模式

### 方式 2: 使用 npm 命令

```bash
# 1. 进入目录
cd mongo-init/local-deploy

# 2. 安装依赖
npm install

# 3. 执行初始化
npm run init              # 普通初始化（保留现有数据）
npm run init:clear        # 清空重建（删除所有数据）
```

## 数据结构说明

### User (tb_user)

```javascript
{
  user_name: String,           // 用户名（唯一）
  password: String,            // 加密密码
  email: String,               // 邮箱（唯一）
  phoneNumber: String,         // 手机号
  student_id: String,          // 学号（唯一）
  avatarUrl: String,           // 头像URL
  balance: Number,             // 账户余额
  roles: [String],             // 角色列表
  runnerProfile: {             // 跑腿员档案（可选）
    status: String,            // ONLINE/OFFLINE/DELIVERING
    rating: Number,            // 评分 (0-5)
    totalOrdersCompleted: Number,
    idCardNumber: String,
    healthCertificateUrl: String,
    isVerified: Boolean,
    runnerSince: Date,
    totalEarnings: Decimal128
  },
  addresses: [{                // 地址列表
    addressId: String,
    name: String,              // 收件人
    building: String,          // 楼栋
    room: String,              // 房间号
    notes: String,             // 备注
    isDefault: Boolean         // 是否默认
  }],
  createdAt: Date,
  updatedAt: Date,
  lastLoginAt: Date
}
```

### Dish (tb_dish)

```javascript
{
  dish_name: String,           // 菜品名称
  restaurant: String,          // 食堂名称
  image_data: String,          // Base64 图片数据
  image_type: String,          // 图片类型 (jpeg/png)
  rating: Number,              // 评分 (0-5)
  price: Number,               // 价格
  description: String,         // 描述
  category: String,            // 分类
  review_count: Number,        // 评价数量
  isAvailable: Boolean,        // 是否可供应
  created_at: Date,
  updated_at: Date
}
```

## 配置文件

### 数据库连接

工具会自动从以下位置读取配置（按优先级）：

1. **backend/MyCampus/src/main/resources/application.properties**
   ```properties
   spring.data.mongodb.uri=${SPRING_DATA_MONGODB_URI:mongodb://localhost:27017/mycampus}
   spring.data.mongodb.database=${SPRING_DATA_MONGODB_DATABASE:mycampus}
   ```

2. **环境变量**
   ```bash
   export SPRING_DATA_MONGODB_URI="mongodb://localhost:27017"
   export SPRING_DATA_MONGODB_DATABASE="mycampus"
   ```

3. **默认值**
   ```
   mongodb://localhost:27017/mycampus
   ```

### 自定义数据

修改 `data/init-data.json` 文件，添加你自己的用户和菜品数据。

## 常见问题

### Q: 如何验证初始化是否成功？

```bash
# 连接 MongoDB
mongosh mycampus

# 查看集合
show collections

# 查看用户数量
db.tb_user.countDocuments()

# 查看菜品数量
db.tb_dish.countDocuments()

# 查看某个用户
db.tb_user.findOne({user_name: "admin"})
```

### Q: 如何重置数据库？

```bash
npm run init:clear
```

或者手动删除：
```bash
mongosh mycampus --eval "db.dropDatabase()"
```

### Q: 如何添加新的测试数据？

1. 复制 `data/init-data.json` 为 `data/my-data.json`
2. 编辑 `my-data.json`，添加你的数据
3. 运行：`node scripts/init-database.js data/my-data.json`

### Q: 密码如何生成？

使用 BCrypt 加密：

```javascript
const bcrypt = require('bcryptjs');
const hash = bcrypt.hashSync('your-password', 10);
console.log(hash);
```
