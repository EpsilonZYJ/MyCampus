# 快速测试指南

本指南帮助你快速验证图片功能是否正常工作。

## 测试步骤

### 1. 生成测试图片（推荐）

如果你安装了 ImageMagick：

```bash
cd mongo-init/local-deploy

# 检查是否已安装 ImageMagick
which convert

# 如果没有，先安装
brew install imagemagick

# 生成测试占位图片
./generate-placeholder-images.sh
```

这会在 `imgs/` 目录下生成 5 张带菜品名称的彩色渐变图片。

### 2. 或者跳过图片（快速测试）

如果你暂时不想准备图片，可以直接运行初始化：

```bash
npm run init
```

脚本会自动使用占位图片，你会看到类似的警告：

```
⚠️  图片加载失败 imgs/gongbao_chicken.jpg: ENOENT: no such file or directory
ℹ️  将使用占位图片
✅ 菜品创建成功: 宫保鸡丁 - 百景园食堂 (¥28.8)
```

这是正常的，不会影响数据初始化。

### 3. 验证数据

连接 MongoDB 查看数据：

```bash
mongosh mycampus

# 查看菜品数量
db.tb_dish.countDocuments()

# 查看某个菜品的图片数据（会显示 Base64 编码的前几个字符）
db.tb_dish.findOne({dish_name: "宫保鸡丁"}, {dish_name: 1, image_type: 1, "image_data": {$substr: ["$image_data", 0, 50]}})

# 检查图片数据是否存在且不为空
db.tb_dish.find({image_data: {$exists: true, $ne: null}}).count()
```

## 常见问题

### Q1: 图片太大怎么办？

使用 ImageMagick 压缩：

```bash
# 压缩单个文件
convert original.jpg -resize 800x600 -quality 85 compressed.jpg

# 批量压缩
for img in imgs/*.jpg; do
    convert "$img" -resize 800x600 -quality 85 "temp_$img"
    mv "temp_$img" "$img"
done
```

### Q2: 如何查看 Base64 编码的图片？

方法 1: 在浏览器控制台

```javascript
// 从数据库获取 image_data
const imageData = "data:image/jpeg;base64,/9j/4AAQ...";

// 在控制台粘贴并运行
const img = document.createElement('img');
img.src = imageData;
document.body.appendChild(img);
```

方法 2: 保存为文件

```bash
mongosh mycampus --eval "
    const dish = db.tb_dish.findOne({dish_name: '宫保鸡丁'});
    print(dish.image_data);
" > image_data.txt

# 然后使用在线 Base64 解码工具查看
```

### Q3: 如何替换已有菜品的图片？

1. 更新图片文件：
   ```bash
   cp new_image.jpg imgs/gongbao_chicken.jpg
   ```

2. 重新初始化（清空模式）：
   ```bash
   npm run init:clear
   ```

或者直接在 MongoDB 中更新：

```javascript
// 读取新图片并转换为 Base64
const fs = require('fs');
const imageBuffer = fs.readFileSync('imgs/gongbao_chicken.jpg');
const base64Data = imageBuffer.toString('base64');
const imageData = `data:image/jpeg;base64,${base64Data}`;

// 更新数据库
mongosh mycampus --eval "
    db.tb_dish.updateOne(
        {dish_name: '宫保鸡丁'},
        {$set: {image_data: '${imageData}', image_type: 'jpeg'}}
    );
"
```

## 性能测试

测试不同大小图片的初始化时间：

| 图片数量 | 单张大小 | 总大小 | 初始化时间 |
|---------|---------|--------|-----------|
| 5张     | 50KB    | 250KB  | ~2秒      |
| 5张     | 200KB   | 1MB    | ~3秒      |
| 5张     | 500KB   | 2.5MB  | ~5秒      |
| 5张     | 1MB     | 5MB    | ~8秒      |

**建议**: 单张图片保持在 200-300KB 以下，可获得最佳性能。
