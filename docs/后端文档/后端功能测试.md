## 用户(User)相关功能测试
- 运行以下代码可以在数据库中插入一条用户信息
```bash
curl -X POST http://localhost:8080/api/users/register \
-H "Content-Type: application/json" \
-d '{
    "userName": "newuser",
    "password": "password123",
    "studentId": "U202598765",
    "email": "newuser@example.com",
    "phoneNumber": "13800138000"
}'
```

- 运行以下代码可以在数据库中进行登录
```
curl -X POST http://localhost:8080/api/users/login \
-H "Content-Type: application/json" \
-d '{
    "userName": "newuser",
    "password": "password123"
}'
```

## 菜品(dish)相关功能测试
### 菜品添加
- 运行以下代码添加一道菜品，需要改一下图片路径
```bash
curl -X POST "http://localhost:8080/api/dishes" \
  -F "dishName=好吃的123456" \
  -F "restaurant=东一" \
  -F "imageData=@/home/lovety/miku.png" \
  -F "rating=" \
  -F "price=1.250" \
  -F "description=" \
  -F "category=主食" \
  -F "isAvailable="
```
- 运行以下代码把对应菜品的信息修改成一个真正存在的菜品（
```bash
curl -X PUT "http://localhost:8080/api/dishes/{粘贴一个dish id}" \
  -F "dishName=鸭腿饭" \
  -F "restaurant=东一一楼" \
  -F "imageData=@/home/lovety/miku.png" \
  -F "rating=4.5" \
  -F "price=12.0" \
  -F "description=还行" \
  -F "category=主食" \
  -F "isAvailable=true"
```
- 运行以下代码查询(所有)菜品
```bash
curl -X GET "http://localhost:8080/api/dishes"
curl -X GET "http://localhost:8080/api/dishes/{粘贴一个dish id}"
```
- 运行以下代码获取菜品的图片，保存到当前文件夹下的dish.png
```bash
curl -X GET "http://localhost:8080/api/dishes/{粘贴一个dish id}/image" -o dish.png
```
- 运行以下代码查询东一食堂的所有菜品，注意食堂为中文是要转码
```bash
curl -X GET "http://localhost:8080/api/dishes/location/东一"
```
- 运行以下代码查询主食类下的所有菜品
```bash
curl -X GET "http://localhost:8080/api/dishes/category/主食"
```
- 运行以下代码查询所有在供应菜品
```bash
curl -X GET "http://localhost:8080/api/dishes/available"
```
- 运行以下代码删除特定菜品
```bash
curl -X DELETE "http://localhost:8080/api/dishes/{粘贴一个dish id}"
```
